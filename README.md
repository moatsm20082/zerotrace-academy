# ZeroTrace Academy Pro V1

Professional Next.js prototype for a cybersecurity academy designed to grow into a real TryHackMe/picoCTF competitor.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase-ready client
- Vercel-ready deployment

## Pages
- Landing page
- Auth
- Dashboard
- Rooms
- Labs
- AI Mentor
- Tournaments
- Pricing
- Admin

## Setup locally
```bash
npm install
npm run dev
```

## Vercel environment variables
```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## Important
This is V1 frontend + Supabase-ready scaffold. Real auth, uploads, submissions, payments, AI, and tournaments become fully functional after Supabase tables/API integrations are added.
